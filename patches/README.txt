
WHAT IS CURRENTLY IN THIS DIRECTORY?
------------------------------------

There is nothing currently in this directory.

WHAT TO PLACE IN THIS DIRECTORY?
---------------------------------

Place patches you generate in this directory.

HOW DO I CREATE A PATCH?
------------------------

Using Git.

1. Edit the contributed module or theme file so that it works how you want.

2.  In the terminal: $git diff path/to/file/form/where/you/are > patches/[module-name]-[issue-description].patch

3. Open the patch and change change /docroot/sites/all/modules/contrib/[module-name]/file.xyz to just file.xyz)
